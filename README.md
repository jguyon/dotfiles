- make (for installing)

- curl (for installing)

- fish

- neovim
- node.js (for coc.nvim)
- alacritty

- sway
- swayidle (for sway)
- swaylock (for sway)
- mako (for sway)
- j4-dmenu-desktop (for sway)
- bemenu (for sway)
- pulseaudio (for sway)
- amixer (for sway)
- i3blocks (for sway)
- Iosevka Term (for alacritty)
- Iosevka Aile (for qutebrowser and sway)

## Install dotfiles

```sh
git clone git@github.com:jguyon/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
make
```
